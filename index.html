Features
div>

Everything your AI agent needs.

🧠
div>

Persistent Memory

Your AI remembers everything across sessions. What's done, what's next, what's blocked. No more re-explaining your project every time.

div>
⚡
div>

Self-Updating

After every task, the AI updates its own memory. Marks items done, adds new links, rewrites priorities. The doc stays current automatically.

div>
📄
div>

Google Docs + Sheets

Uses tools you already have. Google Docs for narrative memory (tasks, context, rules). Google Sheets for structured data (CRM, calendars, tracking).

div>
🌍
div>

Works With Any AI

Bring your own LLM. Claude, ChatGPT, Gemini, LLaMA — the OS is model-agnostic. You provide the brain, we provide the structure.

div>
🛠
div>

4-Part Memory Architecture

SOUL (identity), TODO/DONE (task state), LIVE LINKS (references), WHAT'S NEXT (priorities). A complete operating system in one document.

div>
🏠
div>

Works Everywhere

Plain chat, browser mode, desktop agents, API/MCP setups. Same memory structure, any environment. From free ChatGPT to full automation.
